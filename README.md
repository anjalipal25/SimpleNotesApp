#Simple Notes App

A basic Android app to write and save notes locally using SQLite

## Features

 -Add notes and save them locally

 -prevent saving empty notes

 -Delete notes by long press

 -Local SQLite database storage

 -simple and clean UI

## How to run

 1.clone the repo.
 
 2.open the project in android studio
 
 3.build or run on the android studio emulator
